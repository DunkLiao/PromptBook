# 📚 提示詞管理系統 (PromptBook)

一個純前端、無需後端的 **AI 提示詞管理工具**，所有資料儲存於瀏覽器本地端，開啟即用。

---

## ✨ 功能特色

### 📝 提示詞管理
- **新增 / 編輯 / 刪除**提示詞
- **釘選**重要提示詞，使其永遠排列於最上方
- **Markdown 支援**：內容即時預覽（粗體、程式碼、清單、標題等）

### 📂 資料夾系統
- 建立**多層巢狀資料夾**，靈活分類提示詞
- 編輯或刪除資料夾（刪除時自動將提示詞移至「未分類」）

### 🏷️ 標籤系統
- 為提示詞加上自訂標籤（逗號分隔）
- 側邊欄顯示**熱門標籤雲**，支援多標籤篩選
- 搜尋欄支援 `#標籤` 語法快速查找

### 🔍 全文搜尋
- 即時搜尋標題與內容
- 可同時套用資料夾 + 標籤 + 關鍵字三層篩選

### 🧩 變數模板
- 提示詞內容支援 `{{變數名稱}}` 或 `[變數名稱]` 語法
- 複製時彈出填寫視窗，自動替換後再複製到剪貼簿

### 📦 匯入 / 匯出
- 一鍵**匯出**所有資料夾與提示詞為 JSON 備份檔
- 從 JSON 備份**匯入**，支援「覆蓋」或「附加」兩種模式
- 雙重確認保護的**重設資料庫**功能

### 🌙 深色 / 淺色模式
- 右上角切換按鈕，偏好設定自動記憶

### 💾 本地儲存
- 使用瀏覽器 **IndexedDB** 儲存所有資料，無需帳號或伺服器
- 頁首顯示目前 IndexedDB 儲存空間使用量

## 🗂️ 專案結構

```
PromptBook/
├── index.html      # 主頁面與所有 HTML 結構
├── style.css       # 全域樣式（含深色模式）
└── js/
    ├── db.js       # IndexedDB 資料層（CRUD for 提示詞 & 資料夾）
    └── app.js      # 應用程式邏輯（UI、篩選、匯入匯出等）
```

---

## 🚀 快速開始

此專案為純靜態網頁，**不需要任何安裝或建置步驟**。

1. 下載或 clone 此專案
2. 直接用瀏覽器開啟 `index.html`
3. 開始新增你的第一個提示詞！

> 建議使用現代瀏覽器（Chrome、Edge、Firefox、Safari），需支援 IndexedDB 與 Clipboard API。

## 💡 使用提示

### 變數模板語法

在提示詞內容中使用以下兩種語法定義變數：

```
你是一位 {{角色}} 專家，請用 {{語氣}} 的方式回答以下問題：
[問題內容]
```

點擊「複製」時，系統會自動偵測變數並彈出填寫視窗。

### 資料備份建議

由於資料儲存在瀏覽器本機，建議定期使用**匯出**功能備份 JSON 檔，避免因清除瀏覽器資料而遺失。
